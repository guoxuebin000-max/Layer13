# Layer13 ComfyUI Nodes

This repository contains Layer13 custom nodes for ComfyUI:

- `comfyui-photorealism-enhancer` (Layer13照片增强)
- `comfyui-layer13-depth-of-field` (Layer13｜真实景深（深度图）)
- `comfyui-jpg-convert` (Layer13保存JPG)

## Install

Copy folders into `ComfyUI/custom_nodes/` and restart ComfyUI.

## Notes

All nodes appear under the `Layer13` category in the node list.
